About This Project

A comprehensive machine learning solution for predicting Olympic medal counts, built with modern data science techniques and explainable AI principles.

Project Overview

Objective

Predict the total medal count for each participating country in the 2028 Los Angeles Olympic Games using historical performance data and economic indicators. The project demonstrates end-to-end data science capabilities from data preprocessing to model deployment and visualization.

Key Features

  • ML-powered predictions using Gradient Boosting
  • Interactive dashboard with real-time visualizations
  • What-If scenario simulator for hypothesis testing
  • Model explainability with SHAP values
  • RESTful API for programmatic access

Methodology

1. Data Collection & Preprocessing

Historical Olympic medal data merged with economic indicators (GDP, GDP per capita) and population statistics. Data cleaning handled missing values and standardized formats.

2. Feature Engineering

Created lag features for past performance (4-year Olympic cycles), calculated growth rates, and normalized economic indicators for cross-country comparison.

3. Model Selection & Training

Selected Gradient Boosting Regressor for its ability to capture non-linear relationships and handle mixed data types effectively. Hyperparameters tuned for optimal performance.

4. Model Evaluation

Evaluated using R², RMSE, and MAE metrics. Validated on historical data to ensure predictions align with actual Olympic outcomes.

5. Deployment & Visualization

Built interactive web application with Flask, providing dashboards, API endpoints, and what-if scenario analysis for end-users.
